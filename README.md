# 📱 LogiAid Mobile

Aplicação mobile desenvolvida para otimizar processos logísticos e operacionais através de uma interface moderna, intuitiva e focada em produtividade.

O projeto foi construído com foco em organização arquitetural, escalabilidade e experiência do usuário, aplicando boas práticas de desenvolvimento mobile, componentização e integração com APIs.

---

# 🚀 Tecnologias Utilizadas

- React Native
- TypeScript
- Expo
- React Navigation
- Axios
- Context API
- Async Storage
- REST APIs

---

# 🧠 Objetivos do Projeto

O LogiAid Mobile foi desenvolvido com o objetivo de:

- Centralizar operações logísticas em um único aplicativo
- Melhorar a experiência operacional do usuário
- Facilitar gerenciamento e acompanhamento de processos
- Fornecer uma interface responsiva e moderna
- Aplicar conceitos de arquitetura escalável em aplicações mobile

---

# ⚙️ Funcionalidades

- 🔐 Autenticação de usuários
- 📦 Gerenciamento de operações/logística
- 🔄 Consumo de APIs REST
- 📱 Interface responsiva para dispositivos móveis
- 💾 Persistência local de dados
- 🚦 Controle de estados e navegação
- 🔍 Estrutura modular e componentizada
